Zoom Webinar Report

Free in-browser tool for deduplicated attendee reports. Multi-day, multi-room, country breakdowns — your data never leaves your browser.

Try it now →

Why this exists

Deduplicates rejoins

Zoom's CSV lists every join/leave as a separate row. Drop your CSV — we collapse rejoins by email and time intervals, summing real watch time.

Multi-day & multi-room

Combine sessions across multiple days and parallel rooms (e.g., Main + interpretation). The XLSX shows per-day columns and a single combined view.

RGPD-friendly

Nothing leaves your browser. No upload, no database, no logging. The optional share link contains only aggregated stats — no emails or names.

What you get

  • Formatted XLSX with Summary, Countries, and Users-by-Country sheets (with custom viewing-time thresholds)
  • In-app dashboard: unique attendees, country breakdown, per-day attendance, engagement time histogram, retention curve, peak concurrent viewers
  • Shareable link encoding only aggregated stats — never any personal data
  • Drag-and-drop, multi-file, auto-grouped by date

How it works

1. Drop CSVs

Export your Zoom attendee reports and drop them in. Multiple files = multiple sources or multiple days.

2. Review grouping

Files are auto-grouped by date. Drag any file to a different day to fix the grouping.

3. Get your report

Set the event title and viewing-time thresholds. View the dashboard, download the XLSX, or share a link.

Questions

Does my data ever leave my browser?
No. The CSV is parsed, deduped, and aggregated entirely in JavaScript on your machine. Only the static page assets are loaded from GitHub Pages.
Does it need a paid Zoom plan?
Only the CSV export feature in Zoom is needed (available on most paid plans). No API integration, no OAuth.
Does it work offline?
Once the page is loaded, yes — the processing runs locally. You can refresh while offline as long as the browser cached the assets.
Is the share link safe to send around?
Yes. The URL fragment contains only aggregated stats (country counts, threshold counts, histogram buckets, etc.) — no emails, names, or individual timestamps.
What file format does Zoom export?
The "Attendee Report" CSV from Reports → Usage Reports. UTF-8 with a BOM is supported.
Does it support multiple interpretation rooms?
Yes. Drop both the Main and Chinese (or any interpretation) CSVs together — files are auto-ordered Main → Chinese in the report.
Is the source open?
Yes — on GitHub (MIT licensed).
